# Contributing

Thanks for your interest in this project.

## Scope
This is a personal portfolio project. Contributions are welcome as issues, docs improvements, or focused fixes. For larger changes, open an issue first to discuss scope.

## Development setup
- Follow the README quickstart for local setup.
- Run the lint/format and test commands listed in the README before submitting.

## Pull requests
- Keep changes focused and documented.
- Update README or docs if behavior or setup changes.
- Include screenshots or diagrams when UI or architecture changes.

## Commit messages
Use short, present-tense summaries (e.g., "Add evaluation task").
