# Contributing

Thanks for your interest in improving this platform.

## Development setup
Prereqs:
- Docker and Docker Compose
- Make
- Terraform, Helm, kubectl

Commands:
- `make dev` to start local services
- `make lint` to run formatting and lint checks
- `make test` to run repo structure checks

## Pull requests
- Keep changes focused and documented.
- Update README if behavior or setup changes.
- Include screenshots or diagrams when you change the portal or architecture.

## Commit messages
Use short, present-tense summaries (e.g., "Add drift check script").
