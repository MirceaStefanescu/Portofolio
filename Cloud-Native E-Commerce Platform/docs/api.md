# API Reference

Base URLs (local):
- Product service: http://localhost:8010
- Order service: http://localhost:8020
- Payment service: http://localhost:8030

## Product service
- `GET /health`
- `GET /products`
- `POST /products`
- `GET /products/{product_id}`
- `PUT /products/{product_id}`
- `DELETE /products/{product_id}`

## Order service
- `GET /health`
- `GET /orders`
- `POST /orders`
- `GET /orders/{order_id}`

## Payment service
- `GET /health`
- `GET /payments`
- `POST /payments`
- `GET /payments/{payment_id}`
